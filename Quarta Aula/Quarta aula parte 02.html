Introdução ao CSS


  • CSS in line - é adicionado diretamente a um elemento HTML usando o atributo style
  • Internal CSS - é inserido na própria página HTML, geralmente dentro da tag style> no cabeçalho (head) do documento:
  • CSS Externo - é colocado em um arquivo separado (normalmente com extensão .css) e é vinculado à página HTML usando a tag link no cabeçalho do documento.
